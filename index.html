    GitHub Issues simplified

    GitDone makes GitHub Issues easier to use. It’s a simple browser extension that streamlines the GitHub Issues interface for non-developers — plus adds handy features and shortcuts that project managers and Task Rabbits will love. GitDone was born at the 2015 Mozilla Festival.

    Who is GitDone for?

    • Project Managers
    • Anyone looking for a fast, easy task manager. Or an issue tracker that both technical and non-technical audiences will like and use.
    • Non-technical users. People new to GitHub or GitHub Issues.
